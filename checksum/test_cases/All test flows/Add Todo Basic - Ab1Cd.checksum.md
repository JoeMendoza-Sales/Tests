## Test Steps

1. Clear localStorage and navigate to https://joetodoapp-a03b4.web.app/
2. Verify the empty state message "No todos yet. Add one above!" is displayed
3. Click on the text input field (#todoInput) and type "Buy groceries"
4. Click the "Add" button (.btn-primary)
5. Verify a new todo item appears with text "Buy groceries"
6. Verify the todo item has an unchecked checkbox (.todo-checkbox)
7. Verify the empty state message is no longer visible
