## Test Steps

1. Clear localStorage and navigate to https://joetodoapp-a03b4.web.app/
2. Type "Original task text" in #todoInput and click the "Add" button
3. Verify the todo item with text "Original task text" is visible
4. Click the edit button (.btn-icon with pencil icon) on the todo item
5. Verify the edit form (.edit-form) appears with the current text pre-filled
6. Clear the text input and type "Updated task text"
7. Click the Save button (.btn-save)
8. Verify the todo text now displays "Updated task text"
